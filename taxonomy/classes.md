# Mineral Classes

Minerals are grouped into classes based on their dominant anion or anionic group. The classification below follows the Dana/Strunz system.

| Class | Dominant anion / group | Examples |
|-------|----------------------|----------|
| **Silicates** | SiO₄⁴⁻ | Quartz, feldspar, mica |
| **Sulfides** | S²⁻ | Pyrite, galena, sphalerite |
| **Carbonates** | CO₃²⁻ | Calcite, malachite, dolomite |
| **Oxides** | O²⁻ | Hematite, corundum, magnetite |
| **Sulfates** | SO₄²⁻ | Gypsum, barite, celestine |
| **Halides** | Cl⁻, F⁻ | Halite, fluorite |
| **Phosphates** | PO₄³⁻ | Apatite, turquoise |
| **Native elements** | — | Gold, silver, copper, sulfur |
| **Hydroxides** | OH⁻ | Goethite, gibbsite |
| **Borates** | BO₃³⁻ | Borax, ulexite |
